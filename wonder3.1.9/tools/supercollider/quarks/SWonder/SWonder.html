SWonder Interface to the TU-Berlin WFS-System


Class to connect to CWonder and monitor the OSC-Stream. 

It uses a normalized coordinate system. 

The area inside the room is normaized to an range of -1 to 1. 

The GUI allows you to move sources with drag and drop.

Right klicking a plane wave source allows you to turn it.

Pressing 'a' adds a source.

When a source is touched 't' changes it's type, 'a' changes it's angle and 'r' removes it.


Creation / Class Methods


*new(cWonder)

Creates a new SWonder instance. 

cWonder - NetAddr of cWonder


*connect

Connect to cWonder. 


*disconnect

Disconnect from cWonder. 

*createProject(name)

name - String

*loadProject(name)

name - String

*saveProject


*drawGUI_(bool)

Switch GUI on/off. 


*addSource(sourceID, point)

sourceID - source id starting at 0

point - coordinates of the source position

*removeSource(sourceID)

sourceID - source id starting at 0

*sourcePosition(sourceID, point, dur)

sourceID - source id starting at 0

point - coordinates of the source position

dur - transition time in seconds

*sourcePlane(sourceID, plane)

sourceID - source id starting at 0

plane - bool


*sourceAngle(sourceID, angle)

sourceID - source id starting at 0

angle - angle in degrees


*sourceColor(sourceID, color)

sourceID - source id starting at 0

color - Color


*sourceName(sourceID, name)

sourceID - source id starting at 0

name - Color


*sourceDoppler(sourceID, doppler)

sourceID - source id starting at 0

doppler - bool

*room_(room)

room - Array of points describing the room, H104 is default

*free

disconnect and clean up


Example


a = SWonder( NetAddr("localhost", 58100)); // creating an instance of SWonder

a.connect // connect it to cwonder

a.drawGUI_(true)

a.room_([ -7.7@0, 7.7@0,  10@8.8, 10@25.7, -10@25.7, -10@8.8, -7.7@ 0 ]) // set room to h104 for testing if nt connected

a.sourcePosition(0, 0@0, 5) // put source 0 in the center of the room

a.sourcePosition(0, 0@0.5) // move it half way to the front

a.addSource(1, 0.5@0.2) // add a source directlly to a specified position

a.sourcePlane(1, true) // add a plane wave source

a.sourceAngle(1, 360.rand) // set it's angle

a.sourceName(1, "new name") // rename it

a.sourceColor(1, Color.rand) // different color

a.removeSource(1) // remove it

a.addNextSource

a.addNextSource(0.1 @ 0.7)

a.free

a.room_([ -7.7@0, 7.7@0,  10@8.8, 10@25.7, -10@25.7, -10@8.8, -7.7@ 0 ]) // set room to h104
